# התנהלות שוטפת — שיאים · גרסה 2.0

מערכת ניהול הפרויקטים של **שיאים** (siim.co.il) — שליטה מלאה על כל רעיון ומוצר, מהרעיון הראשוני ועד הגעת הסחורה למחסן.

> "כל רעיון שדובר — חייב להופיע במערכת. אין רעיונות 'בעל פה'. אין מחיקה שקטה. אין רעיונות נעלמים."

## מה יש בגרסה 2.0

- **☀️ מסך "היום"** — מה דורש תשומת לב עכשיו: איחורים, פרויקטים תקועים, רעיונות שמחכים להחלטה, דדליינים השבוע, ומה השתנה מאז הביקור האחרון.
- **💡 אגף רעיונות** — כל רעיון נכנס, הופך לפרויקט בלחיצה, או נדחה לארכיון עם סיבה. לא נמחק לעולם.
- **👤 אגף לקוחות** — לקוח כישות אמיתית: איש קשר, טלפון לחיץ, וכל הפרויקטים והרעיונות שלו במקום אחד.
- **📦 אגף מוצרים וספקים** — הזיכרון הארגוני של המפעלים בסין: WeChat, מחירים, איכות, ואיפה ייצרנו כל דבר.
- **אבטחה אמיתית** — אימות בשרת עם טוקן, סיסמאות מגובבות ומסונכרנות בין מכשירים, הרשאות נאכפות בשרת (מחיקה — רק ליעקב).
- **סנכרון אמיתי** — עדכוני שדה בודד (אין דריסות הדדיות), רענון אוטומטי כל דקה, ותור כתיבות אופליין.
- **נגישות** — עברית פשוטה, טקסט גדול (17px+ ומצב "טקסט גדול"), כפתורי ביטול (Undo) לכל פעולה הרסנית, מובייל-first, PWA.

## מבנה הריפו

```
index.html          — מסך יחיד (SPA)
app.css             — עיצוב Navy/Gold של שיאים
js/config.js        — כתובת השרת (מוטמעת פעם אחת)
js/…                — קוד האפליקציה
backend/Code.gs     — קוד השרת (Google Apps Script) — שמור בריפו!
docs/מדריך-התקנה.md — התקנה צעד-אחר-צעד בעברית
sw.js, manifest.json — PWA
```

## התקנה

ראו [docs/מדריך-התקנה.md](docs/מדריך-התקנה.md) — כ-20 דקות, פעם אחת.

עד שמחברים שרת אמיתי, האפליקציה רצה ב**מצב הדגמה** עם נתוני דוגמה (משתמש `yakov` או `aharon`, סיסמה `1234`) — אפשר להתרשם מהכל מיד.
